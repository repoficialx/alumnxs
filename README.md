# Alumnxs 👨‍🎓👩‍🎓  
![Static Badge](https://img.shields.io/badge/estado-en%20desarrollo-blue)  
![Static Badge](https://img.shields.io/badge/framework-.NET%204.8%20y%20otros-blueviolet)  
![Static Badge](https://img.shields.io/badge/interfaz-WinForms-orange)

**Alumnxs** es una aplicación educativa de gestión de registros de personas (alumnos, usuarios, miembros…) que apuesta por la **simplicidad, accesibilidad y multiplataforma**. Nació en WinForms con .NET Framework, y poco a poco se expande a nuevas tecnologías de interfaz como **WPF, WinUI 3 y .NET MAUI**.

---

## 🎯 Objetivo

Permitir un registro fácil, visual y local de personas mediante una app ligera y adaptable a diferentes entornos y tecnologías de UI.

---

| Versión            | Framework         | Estado actual      | UI         |
|--------------------|-------------------|---------------------|------------|

| Alumnxs (Legacy)   | .NET Framework 3.5| ✅ Funcional         | WinForms  |

> Nota: Las versiones con .NET Core 9 están pensadas para expansión multiplataforma y no contienen aún lógica implementada.

---

## 🧠 Características 

- Registro de personas usando archivos locales (IO.File)
- Interfaz simple con WinForms
- Navegación integrada con `WebBrowser`
- Datos guardados de forma segura y sencilla
- Ideal para profesores, institutos o proyectos educativos

---

## 🚀 Cómo usar

1. Abre el proyecto `.sln` correspondiente en **Visual Studio**
2. Asegúrate de tener el framework correspondiente instalado:
   - .NET Framework 3.5 o 4.7.2 para versiones clásicas
   - .NET Core 9 para la versión normal
3. Compila y ejecuta
