# Alumnxs 👨‍🎓👩‍🎓  
![Static Badge](https://img.shields.io/badge/estado-en%20desarrollo-blue)  
![Static Badge](https://img.shields.io/badge/framework-.NET%204.8%20y%20otros-blueviolet)  
![Static Badge](https://img.shields.io/badge/interfaz-WinForms-orange)

**Alumnxs** es una aplicación educativa de gestión de registros de personas (alumnos, usuarios, miembros…) que apuesta por la **simplicidad, accesibilidad y multiplataforma**.

---

## 🎯 Objetivo

Permitir un registro fácil, visual y local de personas mediante una app ligera y altamente intercompatible

---

## 🧩 Versiones incluidas

| Versión            | Framework         | Estado actual      | UI         |
|--------------------|-------------------|---------------------|------------|
| Alumnxs (main)     | .NET 9            | ✅ Funcional        | WinForms   |
| Alumnxs (Legacy)   | .NET Framework 3.5| ✅ Funcional         | WinForms  |
| Alumn@s (OG)       | .NET Framework 4.7.2| ⚠️ EOS             | WinForms |

---

## 🧠 Características 

- Registro de personas usando archivos locales
- Interfaz simple y nativa con WinForms
- Navegación integrada con `WebBrowser`
- Datos guardados de forma segura y sencilla
- Ideal para profesores, institutos o proyectos educativos

---

## 🚀 Cómo usar

1. Abre el proyecto `.sln` correspondiente en **Visual Studio**
2. Asegúrate de tener el framework correspondiente instalado:
   - .NET F3.5 para Legacy
   - .NET 9 para nuevas versiones
3. Compila y ejecuta
